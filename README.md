# 실습파일 제출 요령

1. topmentor@daum.net 으로 이메일 제출 
2. 이메일 제목은 '[학번-이름] x주차 실습'으로 지정 <br>
   ex) [2000000 홍길동] 14주차 실습
3. 실습파일은 압축해서 첨부파일로 첨부하실 것 
   (압축파일명은 '학번_이름.zip')
